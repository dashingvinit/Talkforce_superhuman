# talkforce-superhuman

## Description

This is the backend for the talkforce superhuman project.

## Tech Stack

- Node.js
- Express
- Langchain
- Pinecone
- MongoDB
- AWS S3

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Testing

```bash
npm test
```

## Deployment

```bash
npm run build
npm run start
```

## API Documentation

```bash
npm run docs
```
